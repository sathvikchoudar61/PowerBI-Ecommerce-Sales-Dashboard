# E-Commerce Sales Analysis Dashboard (Power BI)

This project analyzes e-commerce sales data using Microsoft Power BI.

## Features
- Sales by Category
- Regional Sales Distribution
- Customer Segment Analysis
- Profit Analysis
- Interactive Filters

## Tools Used
- Microsoft Power BI
- DAX
- Power Query

## Dataset
Superstore Sales Dataset
